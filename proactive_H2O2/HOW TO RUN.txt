1. Make sure that power_supply_server.py is running and if not start it

2. Edit the steps file for the power supply channel that you want to use.
   E.g: steps_A1.yaml for channel A1

   Make sure to fill in an appropriate comment and adjust the steps

3. Run the bat file for the power supply channel that you want to use
   E.g: RUN_A1.bat for channel A1

4. In the GUI program, check that the settings and the steps are correct.
   Write start and press enter.
